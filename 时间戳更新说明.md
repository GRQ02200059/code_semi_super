## 📊 时间戳格式

### 格式说明

```
YYYYMMDD_HHMMSS
```

- `YYYY`: 年份 (4位)
- `MM`: 月份 (2位)
- `DD`: 日期 (2位)
- `HH`: 小时 (2位, 24小时制)
- `MM`: 分钟 (2位)
- `SS`: 秒钟 (2位)

### 示例

```
20251105_103045
│││││││││└┴┴─── 秒: 45
││││││││└───── 分: 30
│││││││└────── 时: 10
││││││└─────── 日: 05
│││││└──────── 月: 11
│└┴┴┴────────── 年: 2025
```

**含义**: 2025年11月5日 10:30:45

## 🚀 使用方法

### 1. 直接运行训练

```bash
# 激活环境
conda activate swin

# 运行训练（会自动添加时间戳）
python train_swin_contrastive.py
```

**输出示例**:
```
2025-11-05 10:30:45.123 | INFO | 🏷️  实验名称: swin_semi_contrastive_labeled20pct_20251105_103045
2025-11-05 10:30:45.123 | INFO | ⏰  时间戳: 20251105_103045
```

### 2. 后台运行

```bash
# 后台训练（日志文件也包含时间戳）
nohup python train_swin_contrastive.py > training_$(date +%Y%m%d_%H%M%S).log 2>&1 &
```

**输出文件**: `training_20251105_103045.log`


### 迁移旧实验

如果想给旧实验添加时间戳：

```bash
# 根据文件修改时间添加时间戳
for dir in outputs/*/; do
    if [[ ! $dir =~ [0-9]{8}_[0-9]{6} ]]; then
        # 获取目录修改时间
        timestamp=$(stat -c %y "$dir" | cut -d' ' -f1,2 | tr -d '-:' | tr ' ' '_')
        new_name="${dir%/}_${timestamp}"
        echo "重命名: $dir -> $new_name"
        # mv "$dir" "$new_name"  # 取消注释以执行
    fi
done
```
