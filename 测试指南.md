### 2. 测试单个Checkpoint

#### 使用Python脚本（推荐特定checkpoint）

创建测试脚本 `test_single.py`:

```python
from baseg.modules.semi_supervised_contrastive import SemiSupervisedContrastiveModule
from baseg.datamodules_semi import SemiSupervisedEMSDataModule
from pytorch_lightning import Trainer
from mmengine import Config

# 加载配置
config_path = "outputs/swin_semi_contrastive_labeled20pct_20251105_103330/version_0/config.py"
config = Config.fromfile(config_path)

# 创建数据模块
datamodule = SemiSupervisedEMSDataModule(**config["data"])

# 创建模型
module = SemiSupervisedContrastiveModule(config["model"], ...)

# 测试
trainer = Trainer(accelerator="gpu", devices=1, precision=16)
results = trainer.test(
    module,
    datamodule=datamodule,
    ckpt_path="outputs/.../best-epoch=epoch=20-val_iou=val_iou=0.8288.ckpt"
)

print(f"Test IoU: {results[0]['test_iou']:.4f}")
print(f"Test F1:  {results[0]['test_f1']:.4f}")
```

## 🎯 高级用法

### 1. 批量测试多个实验

创建批量测试脚本 `test_all_experiments.sh`:

```bash
#!/bin/bash
# 测试所有实验

for exp_dir in outputs/*/; do
    if [ -d "$exp_dir/version_0/weights" ]; then
        echo "测试实验: $exp_dir"
        python test_all_checkpoints.py -e "$exp_dir"
        echo "---"
    fi
done

echo "所有实验测试完成！"
```

### 2. 对比不同实验

```python
# compare_experiments.py
import json
from pathlib import Path

experiments = [
    "outputs/swin_semi_contrastive_labeled20pct_20251105_103330",
    "outputs/swin_semi_supervised_improved_labeled20pct_20251104_211619",
]

results = {}
for exp in experiments:
    report_path = Path(exp) / "test_report.json"
    if report_path.exists():
        with open(report_path) as f:
            results[exp] = json.load(f)

# 找出每个实验的最佳结果
for exp, data in results.items():
    successful = [r for r in data if r.get('success')]
    if successful:
        best = max(successful, key=lambda x: x.get('test_iou', 0))
        print(f"{exp}: IoU={best['test_iou']:.4f}")
```

### 3. 生成性能对比图

```python
import matplotlib.pyplot as plt
import json

# 读取测试结果
with open('outputs/.../test_report.json') as f:
    results = json.load(f)

# 提取数据
successful = [r for r in results if r.get('success')]
names = [r['checkpoint'][:30] for r in successful]
ious = [r['test_iou'] for r in successful]

# 绘图
plt.figure(figsize=(12, 6))
plt.bar(range(len(names)), ious)
plt.xticks(range(len(names)), names, rotation=45, ha='right')
plt.ylabel('Test IoU')
plt.title('Checkpoint Performance Comparison')
plt.tight_layout()
plt.savefig('checkpoint_comparison.png')
```

## 🔍 常见问题

### Q1: 如何选择最佳checkpoint？

**A:** 通常选择 **最高IoU** 的checkpoint作为最终模型：
```bash
# 根据报告选择
best_ckpt="outputs/.../weights/iou/best-epoch=epoch=20-val_iou=val_iou=0.8288.ckpt"
```

### Q2: Val IoU vs Test IoU有什么区别？

**A:** 
- **Val IoU**: 验证集性能（训练时保存checkpoint的依据）
- **Test IoU**: 测试集性能（最终评估指标）

通常Test IoU会略低于Val IoU。

### Q3: 为什么要测试多个checkpoint？

**A:** 
1. 验证模型稳定性
2. 找到真正的最佳模型（Val最佳≠Test最佳）
3. 分析训练过程
4. 选择不同场景的最优模型

### Q4: 测试需要多长时间？

**A:** 
- 单个checkpoint: ~2-5分钟
- 全部7个checkpoint: ~15-30分钟
- 取决于测试集大小和GPU性能
