2. ✅ 伪标签学习 (Pseudo-Labeling)

**自动特征**:
- 自动检测Swin Transformer backbone维度

## 🔬 技术细节

### 数据流程图

```
标注数据 [6, 12, 512²]      无标注数据 [6, 12, 512²]
      ↓                           ↓
      │                      ┌─────┴─────┐
      │                      ↓           ↓
      │                   弱增强       强增强
      │                      ↓           ↓
      └──────────┬───────────┴───────────┘
                 ↓
        Swin Transformer Backbone
        (Stage 1-4: 96→192→384→768)
                 ↓
         ┌───────┴────────┐
         ↓                ↓
   Segmentation      Contrastive ⭐
   Predictions       Features
         ↓                ↓
    监督学习        Global Pooling
    伪标签学习        ↓ [B, 768]
    一致性学习      Projection Head
         ↓           ↓ [B, 128]
    分割损失        对比损失 ⭐
         ↓                ↓
         └────────┬───────┘
                  ↓
            Total Loss
```

### 对比学习工作机制

**正样本对** (Positive Pairs):
- 同一图像的弱增强 vs 强增强
- 相同语义的不同view

**负样本对** (Negative Pairs):
- 不同图像之间
- 不同语义内容

**损失函数**:
```python
L_contrastive = -log(
    exp(sim(anchor, positive) / τ) 
    / 
    Σ exp(sim(anchor, all_samples) / τ)
)
```

其中 τ (tau) = temperature = 0.07

### 混合多种对比方式

```python
# 同时使用全局和像素级对比
class MixedContrastiveModule(SemiSupervisedContrastiveModule):
    def _compute_contrastive_loss(self, ...):
        global_loss = self._compute_global_contrastive_loss(...)
        pixel_loss = self._compute_pixel_contrastive_loss(...)
        return 0.7 * global_loss + 0.3 * pixel_loss
```
